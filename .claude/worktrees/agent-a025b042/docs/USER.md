## 🖥️ 대시보드 UX 기대치

### 디자인 레퍼런스: 토스(Toss) 비즈니스 스타일

| 원칙 | 내용 |
|------|------|
| 카드 기반 레이아웃 | 각 기능을 독립적인 카드로 분리 |
| 큰 숫자, 최소 UI | 핵심 지표를 크게, 부가 정보는 숨김 |
| 색상 언어 | 🟢 수익/상승, 🔴 손실/하락 (한국 주식 컨벤션) |
| 반응형 | 모바일 우선 (사용자가 스마트폰으로 주로 확인) |
| 다크모드 | 기본 지원 |

### Fold 위 (스크롤 없이 보이는 영역) 필수 정보

```
┌─────────────────────────────────────────────────────┐
│  알파 (Alpha)              🟢 시스템 정상             │
├─────────────────────────────────────────────────────┤
│  포트폴리오 총액            오늘의 수익              │
│  ₩10,250,000              +₩250,000 (+2.50%)        │
├─────────────┬───────────────────────────────────────┤
│ 보유 종목 3 │  Strategy A ↑ / Strategy B ↑ 일치      │
└─────────────┴───────────────────────────────────────┘
```

### 주요 화면 구성

1. **홈 대시보드**
   - 포트폴리오 요약 카드 (총액, 당일 P&L, 수익률)
   - KOSPI/KOSDAQ 지수 현황
   - 오늘의 시그널 요약 (Strategy A vs B 비교 뷰)
   - 에이전트 상태 바 (5개 에이전트 색상 상태)

2. **시그널 상세 화면**
   - Strategy A: 5개 인스턴스 카드, 토너먼트 순위표, 우승자 하이라이트
   - Strategy B: 토론 요약 카드, 합의 신뢰도 미터, 토론 전문 보기 버튼

3. **포트폴리오 화면**
   - 현재 보유 종목 목록 (수익/손실 색상)
   - 거래 이력 타임라인
   - 성과 차트 (일간/주간/월간, KOSPI 벤치마크 오버레이)

4. **시장 데이터 화면**
   - 종목 검색 및 캔들차트
   - 실시간 시세 업데이트

5. **설정 화면**
   - 전략 블렌드 비율 슬라이더
   - 알림 채널 설정 (Telegram Chat ID)
   - 페이퍼/실거래 모드 전환 (별도 확인 단계)
